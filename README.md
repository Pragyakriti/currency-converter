# 💱 Currency Converter using JavaScript  

A simple and interactive **Currency Converter** built with **HTML, CSS, and JavaScript**. This tool fetches real-time exchange rates using an API and converts currency values instantly.  

---

## 🚀 Features
- 🌐 Real-time currency exchange rates  
- 🔄 Convert between multiple currencies  
- 🖥️ Clean and responsive user interface  
- ⚡ Fast and lightweight  
- 📱 Mobile-friendly design  

---

## 🛠️ Tech Stack
- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (ES6)** – Logic & API Integration  
- **[Exchange Rate API](https://exchangerate.host/)** – Free real-time currency data
